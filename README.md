A drag-and-drop content calendar — a digital replacement for a whiteboard full of coloured sticky notes.

**Upstream app:** <https://github.com/1612elphi/morelater>

## Overview

MoreLater is a visual content calendar built with Next.js. It lets you plan and schedule work by dragging colour-coded chips across a weekly grid, moving them through a five-stage workflow pipeline, and linking related items together.

## Installation

Install from the command line:

```bash
yunohost app install https://github.com/duncangeere/morelater_ynh
```

## What this package provides

- MoreLater running as a Node.js systemd service behind nginx
- SQLite database stored in `/home/yunohost.app/morelater/`
- Automatic backups and restore support
- Log rotation
- URL change support

## Known limitations

- The MCP server (for Claude AI integration) is not included in this package.
- `better-sqlite3` requires native compilation — build tools (`python3`, `make`, `g++`) are installed automatically as dependencies.
- The build step requires ~1 GB RAM. Low-memory servers may struggle.
- The upstream repo does not currently specify a license.
- This package has not been submitted to the YunoHost app catalogue.
